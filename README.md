# 🎮 Neon Breaker

Jogo estilo Brick Breaker desenvolvido em HTML, CSS e JavaScript.

---

## 🎯 Objetivo

Destruir todos os blocos utilizando a bola sem deixar ela cair.

---

## 🕹️ Controles

- ⬅️ Seta esquerda → mover barra
- ➡️ Seta direita → mover barra
- ⏎ ENTER → reiniciar jogo

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Canvas API
